## Контрольные вопросы

### 1. Разница между циклами `for`, `while` и `do-while`
- **for** — используется, когда заранее известно количество повторений.  
- **while** — выполняется, пока условие истинно, может не выполниться ни разу.  
- **do-while** — выполняется **минимум один раз**, условие проверяется после тела цикла.

---

### 2. Когда лучше использовать каждый цикл
- **for** — когда известно точное количество итераций.  
- **while** — когда количество повторов заранее неизвестно.  
- **do-while** — когда код нужно выполнить хотя бы один раз.

---

### 3. Тернарный оператор `? :` в PHP
```php
$результат = ($условие) ? значение_если_true : значение_если_false;
```

###4. Что произойдет, если в do-while условие изначально ложно
Цикл всё равно выполнится один раз, так как проверка условия происходит после выполнения тела цикла.
