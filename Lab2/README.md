**1.Какие способы установки PHP существуют?**

Существует несколько способов установки PHP:
Установка PHP с официального сайта — загрузка архива с сайта php.net, распаковка и ручная настройка переменных среды.
Установка в составе серверного пакета — использование готовых сборок, таких как XAMPP, OpenServer, Denwer, которые включают PHP, веб-сервер и базу данных.
Установка через менеджеры пакетов — применяется в Linux и macOS (apt, yum, brew).
Использование встроенного сервера PHP — подходит для обучения и тестирования.

**2.Как проверить, что PHP установлен и работает?**

В командной строке выполнить команду:
php -v

**3. Чем отличается оператор echo от print?**

| echo                                | print                        |
| ----------------------------------- | ---------------------------- |
| Не является функцией                | Не является функцией         |
| Может выводить несколько аргументов | Выводит только один аргумент |
| Работает быстрее                    | Работает медленнее           |
| Не возвращает значение              | Возвращает `1`               |
| Чаще используется                   | Используется реже            |
